# Tree Plan

**Vue 3 + TypeScript + Vite**

## 项目简介

Tree Plan 是一个基于 Vue3 + TypeScript + Vite 的项目。

## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Compile and Minify for Production

```sh
pnpm build
```

### Lint with ESLint

```sh
pnpm lint
```
